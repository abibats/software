# 华为云部署指南（从零开始）

## 第一步：购买 ECS 服务器

1. 登录 https://www.huaweicloud.com（没有账号就注册，学生可以申请教育优惠）
2. 进入 **控制台** → **弹性云服务器 ECS** → **购买弹性云服务器**
3. 选择配置：

| 配置项 | 推荐选择 |
|--------|----------|
| 规格 | 通用计算型 2vCPUs 2GiB（够用） |
| 镜像 | Huawei Cloud EulerOS 2.0 或 Ubuntu 22.04 |
| 系统盘 | 40GB SSD |
| 安全组 | Sys-WebServer（已放行 80、22） |
| 带宽 | 5Mbit/s |
| 登录方式 | 密码（设一个你能记住的） |

4. 等几分钟创建完成，记下 **公网IP**

## 第二步：SSH 连接服务器

在你的电脑上打开终端（PowerShell 或 Git Bash 都行）：

```bash
ssh root@你的公网IP
# 输入你创建时设的密码
```

如果 Windows 上 ssh 命令不可用，用华为云控制台右侧的 **远程登录** 按钮直接在网页上操作。

## 第三步：服务器初始化

连上服务器后，逐行执行：

> **EulerOS / CentOS 系统：**

```bash
# 更新系统
yum update -y

# 安装必要软件
yum install -y python3 git nginx

# 创建应用目录
mkdir -p /opt/study-seat
cd /opt/study-seat

# 克隆代码
git clone https://github.com/abibats/software.git .
```

> **Ubuntu 系统（如果你选了 Ubuntu）：**

```bash
apt update && apt upgrade -y
apt install -y python3 git nginx
mkdir -p /opt/study-seat
cd /opt/study-seat
git clone https://github.com/abibats/software.git .
```

## 第四步：配置 systemd 服务（开机自启）

```bash
cat > /etc/systemd/system/study-seat.service << 'EOF'
[Unit]
Description=Study Seat Reservation System
After=network.target

[Service]
Type=simple
WorkingDirectory=/opt/study-seat/backend
ExecStart=/usr/bin/python3 server.py
Restart=always
RestartSec=5
Environment=PYTHONUNBUFFERED=1

[Install]
WantedBy=multi-user.target
EOF

# 启动服务
systemctl daemon-reload
systemctl enable study-seat
systemctl start study-seat

# 确认运行
systemctl status study-seat
```

看到 `active (running)` 就是成功了。

## 第五步：配置 Nginx（80端口访问）

> **EulerOS / CentOS：**

```bash
cat > /etc/nginx/conf.d/study-seat.conf << 'EOF'
server {
    listen 80;
    server_name _;

    location / {
        proxy_pass http://127.0.0.1:8000;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
    }
}
EOF

nginx -t && systemctl enable nginx && systemctl start nginx
```

> **Ubuntu：**

```bash
cat > /etc/nginx/sites-available/study-seat << 'EOF'
server {
    listen 80;
    server_name _;

    location / {
        proxy_pass http://127.0.0.1:8000;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
    }
}
EOF

ln -sf /etc/nginx/sites-available/study-seat /etc/nginx/sites-enabled/
rm -f /etc/nginx/sites-enabled/default
nginx -t && systemctl reload nginx
```

## 第六步：验证

浏览器打开 `http://你的公网IP`，应该能看到登录页面。

用 `student1 / 123456` 登录试试。

## 第七步：配置自动部署（GitHub Secrets）

1. 在服务器上生成 SSH 密钥：

```bash
ssh-keygen -t ed25519 -N "" -f /root/.ssh/deploy_key
cat /root/.ssh/deploy_key.pub >> /root/.ssh/authorized_keys
cat /root/.ssh/deploy_key       # 复制输出的私钥（整段）
```

2. 打开 GitHub 仓库 → **Settings** → **Secrets and variables** → **Actions** → **New repository secret**，逐个添加：

| Secret 名称 | 值 |
|-------------|-----|
| `DEPLOY_HOST` | 你的公网IP |
| `DEPLOY_USER` | root |
| `DEPLOY_SSH_KEY` | 刚才复制的私钥 |
| `DEPLOY_PATH` | /opt/study-seat |

3. 添加后，每次 push 到 main 分支就会自动部署到华为云。

## 第八步：截 CI/CD 截图

1. 打开 GitHub 仓库 → **Actions** 标签页
2. 点进最近一次成功的流水线运行
3. 截图每个 Job 的结果（代码检查、构建、测试、部署）
4. 截图放到项目的 `docs/screenshots/` 文件夹里

## 日常操作

```bash
# 手动重启服务
ssh root@你的公网IP
systemctl restart study-seat

# 查看日志
journalctl -u study-seat -f

# 更新代码
cd /opt/study-seat && git pull && systemctl restart study-seat
```
