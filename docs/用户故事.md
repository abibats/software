# 自习座位预约系统 - 用户故事

## Epic 1：学生预约座位

### US-1.1 查询可用座位

**作为** 学生用户，
**我想要** 按条件筛选和搜索可用座位，
**以便** 找到符合我需求的自习座位。

**验收标准：**
- [ ] 可以按关键词搜索座位
- [ ] 可以按自习室筛选
- [ ] 可以按窗边/电源/静音区筛选
- [ ] 组合筛选条件生效
- [ ] 已被预约的座位显示为不可用

### US-1.2 预约座位

**作为** 学生用户，
**我想要** 选择一个可用座位并预约一个时间段，
**以便** 在指定时间使用该座位。

**验收标准：**
- [ ] 可以选择开始时间（必须是未来时间）
- [ ] 可以选择时长（1-4小时）
- [ ] 同一座位同一时段不可重复预约
- [ ] 预约成功后在我的预约中可见
- [ ] 超过最大预约时长（4小时）被拒绝

### US-1.3 取消预约

**作为** 学生用户，
**我想要** 取消已有的预约，
**以便** 释放座位给其他同学使用。

**验收标准：**
- [ ] 可以取消状态为"已预约"的记录
- [ ] 已签到或已过期的预约不可取消
- [ ] 取消后座位恢复可用

### US-1.4 签到

**作为** 学生用户，
**我想要** 在到达自习室后通过输入动态验证码签到，
**以便** 确认我已到席使用座位。

**验收标准：**
- [ ] 签到需输入房间当日动态码
- [ ] 验证码正确才能签到成功
- [ ] 签到后预约状态变为"已签到"
- [ ] 未到预约开始时间不可签到

### US-1.5 查看我的预约

**作为** 学生用户，
**我想要** 查看自己的所有预约记录，
**以便** 了解预约状态和签到情况。

**验收标准：**
- [ ] 显示所有预约记录（已预约/已签到/已取消/已过期）
- [ ] 按时间倒序排列
- [ ] 可以直接在列表中签到或取消

## Epic 2：管理员管理

### US-2.1 管理自习室

**作为** 管理员，
**我想要** 添加、编辑自习室信息，
**以便** 维护系统中的自习室数据。

**验收标准：**
- [ ] 可以新增自习室（名称、楼栋、院系、开放时间）
- [ ] 可以为每个自习室设置每日动态验证码
- [ ] 可以修改自习室状态
- [ ] 学生角色无权操作

### US-2.2 管理座位

**作为** 管理员，
**我想要** 为自习室添加和管理座位，
**以便** 学生可以预约具体的座位。

**验收标准：**
- [ ] 可以新增座位（编号、所属自习室、窗边/电源/静音属性）
- [ ] 可以修改座位状态
- [ ] 座位属性（窗边/电源/静音）可筛选

### US-2.3 系统参数配置

**作为** 系统管理员，
**我想要** 调整系统参数，
**以便** 灵活控制预约规则。

**验收标准：**
- [ ] 可修改最大预约时长
- [ ] 可修改签到提醒时间
- [ ] 可修改迟到提醒时间
- [ ] 可修改自动取消时间
- [ ] 仅 system:config 权限可操作

### US-2.4 用户角色管理

**作为** 系统管理员，
**我想要** 为用户分配角色，
**以便** 控制不同用户的权限范围。

**验收标准：**
- [ ] 可以查看所有用户列表
- [ ] 可以为用户分配角色（学生/管理员/系统管理员）
- [ ] 权限变更立即生效
- [ ] 仅 user:manage 权限可操作

## Epic 3：违规与统计

### US-3.1 自动违约处理

**作为** 系统，
**我想要** 自动处理超时未签到的预约，
**以便** 防止座位被恶意占用。

**验收标准：**
- [ ] 预约开始后15分钟未签到自动取消
- [ ] 自动生成违约记录
- [ ] 违约记录包含用户、预约、原因

### US-3.2 查看违规记录

**作为** 管理员，
**我想要** 查看所有违规记录，
**以便** 了解违约情况。

**验收标准：**
- [ ] 按时间倒序显示所有违约记录
- [ ] 显示用户信息和违约原因
- [ ] 仅 violation:view 权限可查看

### US-3.3 数据统计

**作为** 用户，
**我想要** 在仪表盘看到关键统计数据，
**以便** 了解系统使用情况。

**验收标准：**
- [ ] 管理员看到全局统计（房间数、座位数、预约数、违约数）
- [ ] 学生看到个人统计（我的预约、已签到）
- [ ] 数据实时更新

## Epic 4：智能助手

### US-4.1 智能问答

**作为** 学生用户，
**我想要** 通过自然语言查询座位和预约相关信息，
**以便** 快速获得帮助。

**验收标准：**
- [ ] 可以查询空座情况
- [ ] 可以查询窗边/电源/静音座位
- [ ] 可以查询个人预约
- [ ] 可以了解签到规则和违约规则
- [ ] 支持快捷提问按钮
