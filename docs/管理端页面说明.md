# 管理端前端页面说明

## 管理后台页 (adminPage)

### 自习室管理
- 登记表单：自习室名称、楼栋、院系、开放时间、关闭时间、每日动态码
- 自习室列表：表格展示所有自习室信息，支持编辑

### 座位管理
- 登记表单：选择自习室、座位编号、窗边/电源/静音属性
- 座位列表：表格展示所有座位，按自习室分组

### 预约记录
- 全部预约记录表格：用户、座位、时间、状态
- 管理员可查看所有用户的预约
- 教室管理员可查看全部预约记录

### 系统参数配置
- 参数编辑表单：最大预约时长、签到提醒时间、迟到提醒时间、自动取消时间
- 仅 system:config 权限可见

### 用户角色管理
- 用户列表：用户名、显示名、院系、当前角色
- 角色分配下拉框：系统管理员/教室管理员/学生
- 仅 user:manage 权限可见

## RBAC 菜单控制

| 角色 | 可见菜单 |
|------|----------|
| 学生 (student:use) | 仪表盘、座位预约、智能助手、我的预约 |
| 教室管理员 (room:manage, seat:manage...) | 学生全部 + 管理后台 |
| 系统管理员 (*) | 全部功能 |

## 权限对 API 的控制

- `room:manage`: 可创建/编辑自习室
- `seat:manage`: 可创建/编辑座位
- `reservation:manage`: 可管理所有预约
- `user:manage`: 可管理用户角色
- `system:config`: 可修改系统参数
- `violation:view`: 可查看违规记录
