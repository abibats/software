# 前端页面组件说明

## 学生端页面

### 座位查询页 (studentPage)
- 关键词搜索框：按座位编号或自习室名称模糊搜索
- 筛选面板：自习室下拉框、窗边/电源/静音复选框
- 座位卡片列表：显示座位编号、所属自习室、属性标签（窗边/电源/静音）、状态
- 预约面板：选择开始时间（下拉选择未来小时）和时长（1-4小时）

### 我的预约页 (recordsPage)
- 预约记录表格：显示座位、自习室、时间、状态
- 签到按钮：弹出输入框输入房间动态码
- 取消按钮：确认后取消预约

### 仪表盘 (dashboardPage)
- 统计卡片：我的预约数、已签到数、进行中预约
- 功能入口卡片：快捷导航到各功能页面

### 智能助手 (assistantPage)
- 聊天界面：消息气泡展示
- 快捷提问按钮：空座查询、我的预约、签到规则、违约规则
- 回车发送消息

## 筛选逻辑

前端筛选参数：
- `keyword`: 座位编号或自习室名关键词
- `room_id`: 自习室ID筛选
- `near_window`: 窗边座位
- `has_power`: 有电源座位
- `quiet_zone`: 静音区座位

所有筛选参数通过 `GET /api/seats` 接口传递给后端处理。

## 交互流程

1. 用户登录 → 根据角色显示对应菜单
2. 学生进入座位查询页 → 选择筛选条件 → 浏览可用座位
3. 选择座位 → 选择时间段 → 点击预约 → 提示成功
4. 在我的预约中查看 → 签到时输入动态码 → 完成签到
5. 可取消未开始的预约
