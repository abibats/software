# 华为云部署指南（从零开始）

## 第一步：购买 ECS 服务器

1. 登录 https://.huaweicloud.com （没有账号就注册，学生可以申请教育优惠）
2. 进入 **控制台** → **弹性云服务器 ECS** → **购买弹性云服务器**
3. 选择配置：

| 配置项 | 推荐选择 |
|--------|----------|
| 计费模式 | 按需计费（用完就关，最省钱） |
| 区域 | 华东-上海一（离复旦近） |
| 规格 | 通用计算型 s6.small.1（1核1G，够用） |
| 镜像 | Ubuntu 22.04 公共镜像 |
| 系统盘 | 40GB 高IO |
| 网络 | 默认VPC |
| 安全组 | **新建安全组，入方向放行 80、22 端口** |
| 登录方式 | 密码（设一个你能记住的） |

4. 等几分钟创建完成，记下 **公网IP**

## 第二步：安全组放行端口

控制台 → **VPC** → **安全组** → 找到你创建的安全组 → **入方向规则** → 添加：

| 优先级 | 策略 | 协议 | 端口 | 源地址 |
|--------|------|------|------|--------|
| 1 | 允许 | TCP | 22 | 0.0.0.0/0 |
| 1 | 允许 | TCP | 80 | 0.0.0.0/0 |

## 第三步：SSH 连接服务器

在你的电脑上打开终端：

```bash
ssh root@你的公网IP
# 输入你创建时设的密码
```

## 第四步：服务器初始化

连上服务器后，逐行执行：

```bash
# 更新系统
apt update && apt upgrade -y

# 安装必要软件
apt install -y python3 git nginx

# 创建应用目录
mkdir -p /opt/study-seat
cd /opt/study-seat

# 克隆代码
git clone https://github.com/abibats/software.git .

# 初始化数据库（首次运行会自动创建）
cd backend
python3 server.py &
# 等几秒看到"已启动"后 Ctrl+C 停掉
```

## 第五步：配置 systemd 服务（开机自启）

```bash
cat > /etc/systemd/system/study-seat.service << 'EOF'
[Unit]
Description=Study Seat Reservation System
After=network.target

[Service]
Type=simple
WorkingDirectory=/opt/study-seat/backend
ExecStart=/usr/bin/python3 server.py
Restart=always
RestartSec=5
Environment=PYTHONUNBUFFERED=1

[Install]
WantedBy=multi-user.target
EOF

# 启动服务
systemctl daemon-reload
systemctl enable study-seat
systemctl start study-seat

# 确认运行
systemctl status study-seat
```

看到 `active (running)` 就是成功了。

## 第六步：配置 Nginx（80端口访问）

```bash
cat > /etc/nginx/sites-available/study-seat << 'EOF'
server {
    listen 80;
    server_name _;

    location / {
        proxy_pass http://127.0.0.1:8000;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
    }
}
EOF

# 启用配置
ln -sf /etc/nginx/sites-available/study-seat /etc/nginx/sites-enabled/
rm -f /etc/nginx/sites-enabled/default
nginx -t && systemctl reload nginx
```

## 第七步：验证

浏览器打开 `http://你的公网IP`，应该能看到登录页面。

用 `student1 / 123456` 登录试试。

## 第八步：配置自动部署（GitHub Secrets）

1. 在服务器上生成 SSH 密钥：

```bash
ssh-keygen -t ed25519 -N "" -f /root/.ssh/deploy_key
cat /root/.ssh/deploy_key.pub >> /root/.ssh/authorized_keys
cat /root/.ssh/deploy_key       # 复制输出的私钥（整段）
```

2. 打开 GitHub 仓库 → **Settings** → **Secrets and variables** → **Actions** → **New repository secret**，逐个添加：

| Secret 名称 | 值 |
|-------------|-----|
| `DEPLOY_HOST` | 你的公网IP |
| `DEPLOY_USER` | root |
| `DEPLOY_SSH_KEY` | 刚才复制的私钥 |
| `DEPLOY_PATH` | /opt/study-seat |

3. 添加后，每次 push 到 main 分支就会自动部署到华为云。

## 第九步：截 CI/CD 截图

1. 打开 GitHub 仓库 → **Actions** 标签页
2. 点进最近一次成功的流水线运行
3. 截图每个 Job 的结果（代码检查、构建、测试、部署）
4. 截图放到项目的 `docs/screenshots/` 文件夹里

## 日常操作

```bash
# 手动重启服务
ssh root@你的公网IP
systemctl restart study-seat

# 查看日志
journalctl -u study-seat -f

# 更新代码
cd /opt/study-seat && git pull && systemctl restart study-seat
```
