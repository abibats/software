# 5分钟演示视频录制脚本

> 使用工具：Windows 自带的 `Win + G` 录屏，或 OBS Studio，或 QQ录屏
> 建议分辨率：1920x1080
> 浏览器全屏（F11）后开始录制

---

## 2:30-4:00 核心功能演示（90秒）

### 学生端（45秒）

**操作：**
1. 用 `student1 / 123456` 登录
2. 展示座位查询页面，筛选窗边座位
3. 预约一个座位
4. 展示"我的预约"中新增的记录
5. 点击签到，输入房间验证码

**口述：**
"学生登录后可以查询座位、按条件筛选。选择一个座位进行预约，预约成功后在'我的预约'中可以看到。到达自习室后通过输入房间动态验证码完成签到。"

### 管理端（45秒）

**操作：**
1. 退出登录，用 `admin / 123456` 登录
2. 展示仪表盘统计数据
3. 展示座位管理页面
4. 展示预约记录
5. 展示系统参数配置
6. 展示用户角色管理

**口述：**
"管理员登录后可以看到全局统计数据。可以管理自习室和座位，查看所有预约记录，配置系统参数如最大预约时长，以及管理用户角色权限。系统实现了三级 RBAC 权限控制。"

## 录制注意事项

1. 提前准备好所有账号（admin/123456、student1/123456），已登录状态
2. 关闭浏览器无关标签页和通知
3. 录制前先完整走一遍流程，确保不卡顿
4. 如果有口误或操作失误，可以暂停重新录
5. 录完后用剪映等工具剪辑一下，去掉等待时间
6. 导出格式 MP4，分辨率 1080p
