# 测试与 CI/CD 说明

## 测试框架

使用 Python 标准库 `unittest`，无需安装第三方依赖。

运行命令：
```bash
cd backend
python -m unittest discover -s . -p "test_*.py" -v
```

## CI/CD 流水线

### GitHub Actions 配置 (.github/workflows/ci.yml)

**触发条件：** 每次 push 和 PR

**Job 1: Code quality check (lint)**
- flake8 代码质量检查

**Job 2: Build and verify (build)**
- Python 语法验证
- 前端资源文件检查
- 服务模块加载验证

**Job 3: Automated tests (test)**
- 运行环境：Windows + Python 3.12
- 执行全部 29 个测试用例
- 依赖 lint 和 build 通过

**Job 4: Deploy to Huawei Cloud (deploy)**
- main 分支 push 时自动触发
- SSH 到华为云 ECS 拉取代码并重启服务
- 需配置 GitHub Secrets

**Job 5: Post-deploy smoke test (smoke-test)**
- 部署后对线上接口进行验证
- 需配置 SMOKE_BASE_URL

## 智能助手

### 本地规则匹配（无API时的降级方案）

支持的关键词匹配：
- 空座查询 → 查询当前可用座位数量
- 窗边/电源/静音座位 → 按属性筛选
- 我的预约 → 查询个人预约列表
- 签到规则 → 返回签到流程说明
- 违约规则 → 返回违约处理规则
- 取消预约 → 返回取消操作指引

### MiMo API 集成

当配置了 MiMo API Key 时：
- 发送系统提示词 + 用户问题 + 上下文数据
- 支持同时发送可用座位信息和用户预约信息
- 支持 OpenAI 和 Anthropic 兼容格式
