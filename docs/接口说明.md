# 接口说明

后端地址：

```text
http://127.0.0.1:8000
```

所有接口返回 JSON。除登录外，其余接口需要请求头：

```text
Authorization: Bearer <token>
```

## 1. 登录

```text
POST /api/login
```

请求：

```json
{
  "username": "student1",
  "password": "123456"
}
```

返回：

```json
{
  "token": "...",
  "user": {},
  "permissions": []
}
```

## 2. 当前用户

```text
GET /api/me
```

返回当前登录用户和权限。

## 3. 自习室

```text
GET /api/rooms
POST /api/rooms
PUT /api/rooms
```

新增自习室示例：

```json
{
  "name": "教学楼 B101",
  "building": "第二教学楼",
  "department": "全校",
  "open_time": "07:00",
  "close_time": "22:00",
  "daily_code": "B101"
}
```

## 4. 座位

```text
GET /api/seats
POST /api/seats
PUT /api/seats
```

查询参数：

```text
keyword=图书馆
room_id=1
near_window=1
has_power=1
quiet_zone=1
status=active
```

新增座位示例：

```json
{
  "room_id": 1,
  "code": "1-20",
  "near_window": true,
  "has_power": true,
  "quiet_zone": false
}
```

## 5. 预约

```text
GET /api/reservations
POST /api/reservations
PUT /api/reservations
```

创建预约：

```json
{
  "seat_id": 1,
  "start_time": "2026-05-25T19:00",
  "hours": 2
}
```

取消预约：

```json
{
  "id": 1
}
```

## 6. 签到

```text
POST /api/checkin
```

请求：

```json
{
  "reservation_id": 1,
  "code": "LIB101"
}
```

## 7. 违约记录

```text
GET /api/violations
```

需要管理员权限。

## 8. 统计

```text
GET /api/stats
```

返回自习室、座位、预约、签到和违约数量。

## 9. 用户与角色

```text
GET /api/users
PUT /api/users
GET /api/roles
```

修改用户角色：

```json
{
  "id": 3,
  "role_id": 2
}
```

## 10. 系统参数

```text
GET /api/parameters
PUT /api/parameters
```

修改参数：

```json
{
  "key": "max_hours",
  "value": "4"
}
```

## 11. 智能助手

```text
POST /api/assistant
```

请求：

```json
{
  "message": "帮我找靠窗的座位"
}
```

返回：

```json
{
  "reply": "找到以下可预约座位：..."
}
```
