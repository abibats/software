# 数据库与权限体系设计说明

## 数据库表结构

### roles 表（角色定义）
| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER PK | 自增主键 |
| name | TEXT | 角色名称 |
| permissions | TEXT | JSON数组，存储权限列表 |

### users 表（用户）
| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER PK | 自增主键 |
| username | TEXT UNIQUE | 登录名 |
| password | TEXT | 密码 |
| display_name | TEXT | 显示名 |
| department | TEXT | 院系 |
| role_id | INTEGER FK | 关联角色 |

### rooms 表（自习室）
| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER PK | 自增主键 |
| name | TEXT | 自习室名称 |
| building | TEXT | 楼栋 |
| department | TEXT | 院系 |
| open_time | TEXT | 开放时间 |
| close_time | TEXT | 关闭时间 |
| status | TEXT | 状态 |
| daily_code | TEXT | 每日动态码 |

### seats 表（座位）
| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER PK | 自增主键 |
| room_id | INTEGER FK | 所属自习室 |
| code | TEXT | 座位编号 |
| near_window | INTEGER | 窗边 0/1 |
| has_power | INTEGER | 电源 0/1 |
| quiet_zone | INTEGER | 静音区 0/1 |
| status | TEXT | 状态 |

### reservations 表（预约）
| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER PK | 自增主键 |
| user_id | INTEGER FK | 用户 |
| seat_id | INTEGER FK | 座位 |
| start_time | TEXT | 开始时间 |
| end_time | TEXT | 结束时间 |
| status | TEXT | 状态(reserved/checked_in/cancelled/expired) |
| checked_in_at | TEXT | 签到时间 |

### violations 表（违规）
| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER PK | 自增主键 |
| user_id | INTEGER FK | 用户 |
| reservation_id | INTEGER FK | 预约 |
| reason | TEXT | 违约原因 |
| created_at | TEXT | 创建时间 |

### parameters 表（系统参数）
| 字段 | 类型 | 说明 |
|------|------|------|
| key | TEXT PK | 参数键 |
| value | TEXT | 参数值 |
| label | TEXT | 显示名 |

## RBAC 权限体系

### 角色定义

| 角色 | 权限 |
|------|------|
| 系统管理员 | ["*"] 全部权限 |
| 房间管理员 | ["room:manage", "seat:manage", "reservation:view", "reservation:manage", "violation:view"] |
| 学生 | ["student:use"] |

### 权限检查机制

`has_perm(user, perm)` 函数：
1. 获取用户角色对应的权限列表
2. 如果权限列表包含 `"*"`，返回 True（全部权限）
3. 否则检查目标权限是否在列表中

### Token 鉴权

1. 登录成功后生成随机 Token，存入内存字典 `TOKENS`
2. 后续请求通过 `Authorization: Bearer <token>` 携带
3. 每个受保护的 API 在处理前验证 Token 有效性
4. 从 Token 映射到用户信息，获取角色和权限
